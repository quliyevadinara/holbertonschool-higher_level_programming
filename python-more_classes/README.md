# Rectangle Class Documentation

## Overview
The `Rectangle` class is designed to represent rectangular shapes. It provides methods to create rectangles, calculate their area and perimeter, and perform other geometric operations.

## Attributes
- `width`: Represents the width of the rectangle. Must be a positive integer.
- `height`: Represents the height of the rectangle. Must be a positive integer.

## Initialization
```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height
