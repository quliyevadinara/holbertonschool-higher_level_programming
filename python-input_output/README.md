# Python - Input/Output

This project contains Python scripts and modules for file operations, JSON serialization, and data manipulation.

## Requirements

- Python 3.8.5+
- Ubuntu 20.04 LTS
- pycodestyle (PEP 8 compliance)

## Files

### File Operations
- **0-read_file.py** - Read and print UTF-8 text files
- **1-write_file.py** - Write strings to files and return character count
- **2-append_write.py** - Append strings to files

### JSON Operations
- **3-to_json_string.py** - Convert objects to JSON strings
- **4-from_json_string.py** - Convert JSON strings to objects
- **5-save_to_json_file.py** - Save objects to JSON files
- **6-load_from_json_file.py** - Load objects from JSON files
- **7-add_item.py** - Script to add items to a JSON list file

### Class and Object Serialization
- **8-class_to_json.py** - Convert class instances to dictionaries
- **9-student.py** - Student class with JSON serialization
- **10-student.py** - Student class with filtered JSON output
- **11-student.py** - Student class with reload functionality

### Algorithm
- **12-pascal_triangle.py** - Generate Pascal's triangle

## Usage

All files are executable Python scripts that can be run directly:

```bash
./0-read_file.py
./1-write_file.py
./7-add_item.py "item1" "item2"
