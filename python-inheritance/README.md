# Python - Inheritance

This project contains implementations of Python classes and functions that demonstrate object-oriented programming concepts, with a focus on inheritance and object introspection.

## Tasks

- **0-lookup.py** - Returns list of attributes and methods
- **1-my_list.py** - MyList class with print_sorted method
- **2-is_same_class.py** - Check exact class type
- **3-is_kind_of_class.py** - Check instance or inheritance
- **4-inherits_from.py** - Check indirect inheritance
- **5-base_geometry.py** - Empty BaseGeometry class
- **6-base_geometry.py** - BaseGeometry with area() method
- **7-base_geometry.py** - BaseGeometry with integer_validator
- **8-rectangle.py** - Rectangle class
- **9-rectangle.py** - Rectangle with area and __str__
- **10-square.py** - Square class
- **11-square.py** - Square with custom __str__
